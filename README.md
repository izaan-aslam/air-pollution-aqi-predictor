# Global Air Quality Prediction System

Machine learning-based regression system to predict global Air Quality Index (AQI) values using pollutant data (CO, NO₂, O₃, PM2.5). The project applies data preprocessing, feature engineering, and multiple regression models to estimate AQI for environmental monitoring.

## Features
- AQI value prediction using regression models
- Data preprocessing and feature engineering
- Model evaluation using MAE, MSE, and R² score
- Works on global air pollution dataset (100+ countries)

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Dataset
Global Air Pollution Dataset from Kaggle (23,000+ records)

## Goal
To provide an AI-based solution for predicting air quality and supporting environmental decision-making.
